How to Use

Smyle Mouse is a service to control computer
using Face and Voice.

Open the mouth

Give control of mouse to the application by opening the mouth.

Move the face

Control the mouse movements by moving the face to left and right.

Blink the Eyes

Blink Left Eye for left click and Right Eye for right click of the mouse.

Move dir parameter

Control mouse movements by speaking like : Move Right 40

Scroll dir parameter

Scroll mouse by speaking like : Scroll Up 400

Left/Right Click

Control mouse Clicking by speaking like : Left Click

Type this text

Control keyboard typing by speaking like : Type this Google

Press key alphabet

Control keyboard letter typing by speaking like : Press key A

Hold/Release key keyname

To Hold/Release the key speak like : Hold/Release delete

How to Visualize

Instruction to move the cursor in horizontal direction is : (move left/right percentage of screen width )
Instruction to move the cursor in vertical direction is : (move up/down percentage of screen height )
